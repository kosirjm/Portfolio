<p style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 15pt; font-family: Arial; color: #050505; background-color: transparent; font-weight: 400; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;</span><span style="font-size: 12pt; font-family: Arial; color: #050505; background-color: transparent; font-weight: 400; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;">&nbsp;While working at PAKO INC. &nbsp;I was tasked many different jobs and responsibilities. &nbsp;The biggest project I encountered while working at this company was a file sharing website. &nbsp;I was given the responsibility of creating a file sharing website for PAKO Inc.&rsquo;s employees and clients to upload and download proprietary blueprints and other work files. &nbsp;Previously the sending of files could be a long and tedious task. Since PAKO Inc. had proprietary and government contracts simply emailing the files was not an option due to strict protocol and regulation. &nbsp;Therefor this website was created.</span></p>
<p style="line-height: 1.38; margin-top: 0pt; margin-bottom: 0pt;"><span style="font-size: 12pt; font-family: Arial; color: #050505; background-color: transparent; font-weight: 400; font-variant: normal; text-decoration: none; vertical-align: baseline; white-space: pre-wrap;"> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;For this project I was the only team member. &nbsp;I researched, presented, developed, and trained people on this website. &nbsp;To start this project, I had no other previous knowledge of website development, just some intermediate level programming abilities. &nbsp;I taught myself basic php, javascript, html, mysql, css, and other needed topics. After trial and error and much research a framework to work around was decided on, Ajaxplorer. &nbsp;Once this decision was made a WAMP setup was installed on our server. I set up and installed the needed modules and applications onto the server. Once this was done I began development. &nbsp;Many things changed and were added as the process went on. I continuously presented my work and listened to feedback from my supervisors, which included head of IT, the company CEO, the company owner and many more. &nbsp;With this I accepted the feedback and fixed the issues or added the features which were desired. Though in some circumstances because of time constraints and feasibility certain features were not in the realm of possibility and I had to explain why these ideas could not be included at the time. </span></p>
